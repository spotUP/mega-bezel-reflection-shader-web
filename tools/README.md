# Tools

This folder contains developer tooling and support assets that are not part of the app runtime.

## Shader Tooling
Located under `tools/shader/`.

- `tools/shader/scripts/`: Helper scripts for toggles, rendering, and setup
- `tools/shader/tests/`: CLI checks and debug runs for shader compilation and rendering
- `tools/shader/html/`: Minimal HTML pages for isolated shader validation
- `tools/shader/snippets/`: GLSL snippet references used during debugging
- `tools/shader/assets/`: Reference images used for shader development
- `tools/shader/artifacts/`: Generated logs, reports, and screenshots (gitignored)
